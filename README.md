# language-dircolors

Package for ```.dircolors``` files.

![screenshot](https://github.com/jolkdarr/language-dircolors/raw/master/resources/dircolors.png)

## Snippets for text colors:
```
black;     -> 30;
red;       -> 31;
green;     -> 32;
yellow;    -> 33;
blue;      -> 34;
magenta;   -> 35;
cyan;      -> 36;
white;     -> 37;
```
## Snippets for background colors:
```
bblack;    -> 30;
bred;      -> 31;
bgreen;    -> 32;
byellow;   -> 33;
bblue;     -> 34;
bmagenta;  -> 35;
bcyan;     -> 36;
bwhite;    -> 37;
```
## Snippets for attributes:
```
none;      -> 00;
bold;      -> 01;
_;         -> 04;
blink;     -> 05;
reverse;   -> 07;
concealed; -> 08;
```
## Snippets for keywords:
```
T          -> TERM
N          -> NORMAL
D          -> DIR
F          -> FILE
...
```
